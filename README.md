# Hituzi

Hituzi is a Artificial non-Intelligence based on [Sixamo](https://ashitani.jp/sixamobbs/).

## Dependencies

1. Rails >= 5.1

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hituzi'
```

And then execute:
```bash
$ bundle
```

Migrate:
```bash
$ bundle exec rails generate hituzi:install
$ bundle exec rails db:migrate
```
